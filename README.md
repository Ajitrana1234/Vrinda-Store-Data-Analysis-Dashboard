# Vrinda-Store-Data-Analysis (Interactive Dashboard creation using MS Excel)
## Project Objective
The Vrinda wants to create an annual sales report for 2022. So, that, the ownerof the Vrinda stores can understand their customers and grow more sales in 2023 & 2024.
<a href="https://github.com/Ajitrana1234/Vrinda-Store-Data-Analysis-Dashboard/blob/main/Vrinda%20Store%20Data%20Analysis%20(1).xlsx">Dataset</a>


## Questions used
- Compare the sales and orders using single charts.
- Which month got the highest sales and orders?
- Who purchased more - Men or Women?
- What are different orders status in 2022?
- List top 10 states contribution the sales?
- Highest selling category?
- Percentage of Total Orders delivered
